# Theme System - Quick Start Guide 🎨

## How to Use the Theme System

### For Users

#### Switching Themes
1. Look for the **🎨 Experience Mode** button in the top-right corner
2. Click it to open the theme selector
3. Choose from 5 available themes:
   - **⚡ Cyberpunk** - Futuristic neon aesthetic (Default)
   - **💎 Glass** - Modern glassmorphism design
   - **🖥 Terminal** - Developer console style
   - **🟢 Matrix** - Digital rain and code streams
   - **⚪ Minimal** - Clean professional mode
4. Theme switches instantly with smooth animation
5. Your choice is saved automatically

#### Theme Persistence
- Your selected theme is remembered
- Refresh the page - theme stays
- Close browser - theme persists
- Works across sessions

## 🆘 Troubleshooting

### Theme not switching?
- Check if JavaScript is enabled
- Try refreshing the page
- Clear browser cache

### Theme not saving?
- Check localStorage permissions
- Try incognito mode
- Check browser settings

### Animations laggy?
- Close other browser tabs
- Reduce browser extensions
- Switch to a simpler theme (Minimal)
