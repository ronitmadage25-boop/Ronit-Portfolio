# Project Background Images

## CompressX Background

**File needed:** `compressx-bg.png`

**Instructions:**
1. Save the CompressX background image (the one with PNG, JPG, PDF icons and the compression machine) 
2. Rename it to: `compressx-bg.png`
3. Place it in this folder: `public/images/`

The image will automatically appear as the background for the CompressX project card with 80% transparency.
